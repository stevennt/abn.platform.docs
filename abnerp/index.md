# ABNERP - Hệ thống Hoạch định Nguồn lực Doanh nghiệp

Chào mừng bạn đến với ABNERP - giải pháp **Hoạch định Nguồn lực Doanh nghiệp (ERP)** mã nguồn mở toàn diện, được xây dựng trên nền tảng Frappe Framework. ABNERP giúp doanh nghiệp vừa và nhỏ quản lý toàn bộ hoạt động vận hành trên một nền tảng thống nhất.

## Tính năng chính

ABNERP cung cấp hơn 12 mô-đun nghiệp vụ được tích hợp chặt chẽ:

| Mô-đun | Mô tả |
|---|---|
| **Kế toán** | Quản lý sổ sách kế toán, báo cáo tài chính, thuế |
| **Quản lý Kho** | Theo dõi tồn kho, nhập/xuất, kiểm kê |
| **CRM** | Quản lý quan hệ khách hàng, cơ hội, quy trình bán hàng |
| **Bán hàng** | Quản lý đơn hàng, báo giá, hóa đơn bán hàng |
| **Mua hàng** | Quản lý đơn mua hàng, yêu cầu mua hàng, nhà cung cấp |
| **Sản xuất** | Quản lý lệnh sản xuất, định mức nguyên vật liệu (BOM), lập kế hoạch sản xuất |
| **Quản trị Nhân sự (HRM)** | Quản lý nhân viên, chấm công, lương, nghỉ phép |
| **Hỗ trợ** | Quản lý ticket hỗ trợ khách hàng |
| **Quản lý Chất lượng** | Quy trình kiểm soát chất lượng |
| **Dự án** | Quản lý dự án, nhiệm vụ, thời gian |
| **Tài sản** | Quản lý tài sản cố định và bảo trì |
| **Trang web** | Quản lý nội dung website, cửa hàng trực tuyến |
